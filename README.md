# Health HUD CE

Health HUD CE addon by chronicboy31

Displays health above entities in Better Than Wolves CE 3.0.0

## Building

1. Run `install.bat` (Windows) or `install.sh` (Linux/Mac) to set up the development environment
2. Run `.\gradlew.bat build` (Windows) or `./gradlew build` (Linux/Mac)
3. The compiled mod JAR will be in `build/libs/health-hud-ce-1.0.1.jar`

## Installation

Copy `build/libs/health-hud-ce-1.0.1.jar` to your Better Than Wolves `mods` folder.

## Features

- Green health bar with transparency
- Health text with decimals (shown only when needed)
- Text extends above the bar for better visibility
- Only visible when entities are not obstructed by blocks
- Thin text outline for maximum readability
