# Health HUD CE

**Health HUD CE** is a client-side addon for **Better Than Wolves CE 3.0.0+** that adds a clean, unintrusive health indicator above entities.

Knowing when to engage or retreat is critical in Better Than Wolves. This mod provides clear, elegant visual feedback on entity health without cluttering your screen or feeling out of place.

## ✨ Features

- **Dynamic Health Bar:** A minimalistic green health bar that features a clean, transparent design so it doesn't block your view of the action.
- **Precise Health Values:** Displays exact numerical health text (including decimals when needed) perfectly centered within the bar for quick and precise reading.
- **Maximum Readability:** The health text features a thin, dark outline and renders at full brightness, ensuring it remains perfectly legible even in pitch-black environments like deep underwater or unlit caves.
- **Smart Line-of-Sight:** The HUD is intelligently hidden when an entity is obstructed by blocks, ensuring a fair and immersive experience.
- **Toggle Keybind:** Easily enable or disable the HUD on the fly with a dedicated hotkey (default **H**).

## 📥 Installation

1. Download the latest `.jar` file from the versions tab.
2. Drop it into your Better Than Wolves `mods` folder.
3. Launch the game and enjoy!
